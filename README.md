# Etch-a-Sketch

A browser-based drawing application built with JavaScript, HTML, and CSS as part of The Odin Project.

## Features

- Dynamic grid generated with JavaScript
- Draw by hovering over squares
- Custom grid sizes up to 100×100
- Responsive Flexbox layout
- Input validation for grid resizing

## Technologies

- HTML
- CSS
- JavaScript
- Flexbox

## What I Learned

- DOM manipulation
- Event listeners
- Dynamic element creation
- User input validation
- Flexbox layouts
